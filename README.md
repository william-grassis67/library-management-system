# Library Management System

A simple library management system developed in Java. This project was created to practice object-oriented programming, collections, and CRUD operations in a console application.

## Features

- Register books
- List all books
- Search for a book by title
- Borrow a book
- Validate user input

## Technologies

- Java
- IntelliJ IDEA
- Collections (ArrayList)

## Project Structure

```
src/
├── data/
├── model/
├── service/
└── Main.java
```

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/SEU-USUARIO/library-management-system.git
   ```

2. Open the project in IntelliJ IDEA.

3. Run the `Main.java` file.
